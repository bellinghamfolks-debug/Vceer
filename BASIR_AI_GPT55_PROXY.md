# تحديث بصير AI: ربط GPT-5.5 عبر وسيط آمن

## ما الذي تغير؟

تم تحويل نسخة بصير AI من واجهة محلية مبدئية إلى نسخة تدعم الذكاء الاصطناعي الحقيقي عبر خادم وسيط.

## الميزات المضافة في التطبيق

1. زر: **اسأل بصير GPT-5.5**.
2. زر: **إعداد وسيط GPT-5.5**.
3. إدخال رابط الوسيط من داخل التطبيق.
4. إدخال رمز تطبيق اختياري `BASIR_APP_TOKEN`.
5. تحليل المستندات عبر GPT-5.5 عند توفر الوسيط.
6. الترجمة السياقية عبر GPT-5.5 عند توفر الوسيط.
7. تحليل المنتج نصيًا عبر GPT-5.5 عند توفر الوسيط.
8. تحليل المشهد النصي عبر GPT-5.5.
9. الرجوع التلقائي للتحليل المحلي عند عدم ضبط الوسيط.
10. حفظ نتائج الذكاء السحابي في السجل النصي المحلي.

## ما الذي لم يكتمل بعد؟

هذه النسخة لا ترسل صورة الكاميرا نفسها إلى GPT-5.5 بعد. تم تجهيز البنية النصية أولًا حتى ينجح البناء ويعمل APK بثبات. المرحلة التالية هي:

1. التقاط صورة من الكاميرا داخل التطبيق.
2. تحويل الصورة إلى Base64.
3. إرسالها إلى الوسيط.
4. جعل الوسيط يستدعي نموذجًا متعدد الوسائط.
5. إرجاع وصف بصري دقيق مناسب للمكفوفين.

## لماذا لا نضع OpenAI API Key داخل التطبيق؟

لأن مفتاح API داخل APK يمكن استخراجه بسهولة نسبيًا. التصميم الصحيح هو أن يحتفظ الخادم بالمفتاح، والتطبيق يرسل الطلبات إلى الخادم فقط.

## رابط الوسيط داخل التطبيق

بعد نشر الخادم، افتح التطبيق:

الإعدادات → وسيط GPT-5.5 → رابط الوسيط

ضع الرابط بهذا الشكل:

```text
https://your-domain.com/api/basir
```

ثم ضع رمز التطبيق إن فعلته في الخادم، واضغط اختبار.
