# Basir AI 0.3.0 - Vision + Memory Build

هذه الدفعة تبني فوق نفس مشروع BasirAI_GPT55، وتضيف أول طبقة حقيقية من الذكاء العملي داخل التطبيق.

## إضافات Android

1. مختبر الذكاء
   - تحليل صورة حقيقي عبر وسيط GPT-5.5.
   - توليد وصف بديل مفصل للصور Alt Text.
   - شرح لقطات الشاشة للمكفوفين.
   - تحويل النصوص إلى بطاقات مذاكرة.
   - تحليل قانوني آمن.
   - تحليل صحي آمن.
   - تجهيز رد مناسب على الرسائل.

2. تحليل الصور
   - اختيار صورة من المعرض.
   - تحويلها إلى Base64 داخل التطبيق.
   - إرسالها إلى الخادم الوسيط، وليس مباشرة إلى OpenAI.
   - عدم حفظ الصورة تلقائيًا داخل التطبيق.
   - حد مبدئي لحجم الصورة داخل التطبيق: 7 ميجابايت تقريبًا.

3. الأرشيف والذاكرة
   - حفظ نتائج الذكاء في أرشيف مستندات محلي.
   - عرض آخر المستندات المحفوظة.
   - حفظ أشخاص وملاحظات عنهم محليًا.
   - حفظ منتجات أو أدوية وملاحظات عنها محليًا.
   - حفظ أماكن وملاحظات وصول أو عوائق.
   - حذف كل البيانات المحلية من الإعدادات.

4. تحسين الوصول
   - أوصاف TalkBack للأزرار الجديدة.
   - نتائج منظمة نصيًا وصوتيًا.
   - أوامر صوتية جديدة: مختبر، ذكاء، صورة، أرشيف، ذاكرة.

## إضافات الخادم الوسيط

1. دعم الصور
   - يقبل `image_base64` و `mime_type`.
   - يرسل الصورة إلى Responses API بصيغة input_image.
   - يدعم JPEG و PNG و WebP.

2. تحسين التعليمات
   - تعليمات متخصصة حسب المهمة: صورة، لقطة شاشة، قانون، صحة، ترجمة، دراسة، مستندات.
   - منع ادعاء التشخيص أو الملاحة اللحظية أو التعرف على هوية الأشخاص.

3. حدود أمان مبدئية
   - حد JSON قابل للضبط.
   - حد حجم الصورة Base64.
   - Rate limit بسيط لكل IP.
   - استمرار منع وضع مفتاح OpenAI داخل APK.

## ملفات مهمة

- تطبيق Android: `app/src/main/java/com/basir/ai/MainActivity.java`
- الخادم الوسيط: `server/index.js`
- إعدادات الخادم: `server/.env.example`
- Workflow بناء APK: `.github/workflows/build-apk.yml`
